Para analizar la relación entre variables continuas se usa el scatterplot. 
Los boxplot se usan para ver la distribución de una variable continua (mediana, cuartiles, outliers) comparada entre categorías de una variable categórica.
Los gráficos de barras se usan principalmente para variables categóricas, pero no necesariamente “comparar categóricas entre sí” directamente, sino mostrar frecuencias (cuántos hay de cada categoría), o comparar una medida agregada (promedio, suma) de una variable numérica entre categorías.

SCATTERPLOTS

Ya que vimos que la colinealidad entre distancia de vuelo y el tiempo estimado de vuelo es bastante alta, es redundante evaluar la relación entre las demás variables con ambas variables. Por esto, elegimos una de las dos para hacer las futuras comparaciones, en nuestro caso eligiremos tiempo estimado de vuelo.
